
# 🎧musicboard
 ![Main](https://github.com/nahyunkim123/musicboard/assets/142788257/c2def617-ec7b-4f4a-940f-8f552aba6930)
  <br>
  다양한 음악 취향을 공유하고 싶어하는 사용자들을 위해 제작한 웹사이트입니다!


## 메인

 **회원가입 / 로그인**  
 **글 작성 / 수정 / 삭제**  
 **관리자 페이지**
